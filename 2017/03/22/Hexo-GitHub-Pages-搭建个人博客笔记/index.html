Hexo+GitHub Pages搭建个人博客笔记

Hexo+GitHub Pages搭建个人博客笔记

前言

不久之前加入到Stormzhang的小密圈, 张哥一直强调写作对人生发展的重要性, 慢慢地越来越多密友也踏上了规范标准的写作之路。

对于我来说的话, 并不是想要跟上写作的风气, 毕竟我从小到大的表达能力都不好, 语文写作业仅仅是平均分边缘, 我只是真心觉得写博客对学习过程做一个总结会对自己的学习有很大帮助。

我是接触了Android半年左右的小菜鸟, 学习的时候有遇到一些不懂的地方, 然后在网上都能找到很多人的博客的解决方法, 然后我就一个个收藏。

这样的坏处是:

1、 博客并不是你写的, 对过程印象不会很深刻, 可能只能一时刻解决问题;
2、 内容太分散, 查找起来很麻烦。

因此, 还是自己搭建一个个人博客吧！

如题所示, 本文是Hexo+GitHub Pages搭建个人博客的教程, 直接上手吧!

一、环境配置

使用Hexo之前需要系统中安装Node. js以及Git, 这两个的安装直接从官网上下载即可.

Node. js下载地址：
https://nodejs.org/en/download/
Git下载地址：https://git-scm.com/download/

我的电脑是Windows系统, 很是方便。
