自定义 View

=========================================

自定义 View 的方式：

1、对现有的控件进行扩展

2、通过组件来实现新的控件

3、继承 View 来实现全新的组件

继承 View 类

1、确定控件所需要的参数

2、attrs.xml保存控件所需参数

(XML声明控件时导入, 非必要)

3、获取attrs.xml文件

在java文件中，用typeArray=ccontext.obtainStyledAttributes(attrs, R.styleable.MyView); 来获取attrs中的参数

4、调用typeArray.recycle();

5、onMeasure测量方法：

1、对View测量，改变View大小

2、measureheight:是布局内容的高度

3、height：是布局（view）的高度

模式：EXACTLY(精确值)，AT_MOST(上限至多某个值)，UNSPECIFIED(无限制：ScrollView、ListView)

测量值：MeasureSpec：measureheight、measurewidth

4、setMeasureDimension()

5、requestLayout()

6、onDraw()绘制

1、绘制内容区域

2、Invalidate()UI线程，postInvalidate()可以在子线程中调用；

3、Canvas.drawXXX; point/line/circle

4、Traslate/rotate/scale/skew

5、Save()、restore() 保存、返回画布

7、事件分发(ViewGroup)

  • 外部拦截：
    在ViewGroup的dispatchTouchEvent中判断用户是不是想显示菜单，如果是，则在onInterceptTouchEvent(ev)拦截子View的事件；自己进行处理onTouchEvent。
  • 内部拦截：子View可以通过调用getParent().requestDisallowInterceptTouchEvent(true); 阻止ViewGroup对其MOVE或者UP事件进行拦截。

Paint属性

  • 1、setAntiAlias(); //设置画笔的锯齿效果

  • 2、setColor();//设置画笔的颜色

  • 3、setARGB(); //设置画笔的ARGB值

  • 4、setAlpha(); //设置画笔的Alpha值

  • 5、setTextSize(); //设置字体大小

  • 6、setStyle(); //设置画笔风格（空心、实心）

  • 7、setStrokeWidth(); //设置空心边框的宽度

  • 8、getColor()；//获取画笔的颜色

Canvas属性

  • canvas.drawLine();
  • canvas.drawRect();
  • canvas.drawCircle();
  • canvas.drawText();
  • canvas.drawBitmap();

最后：
1、onSaveInstanceState、
onRestoreInstanceState保存和恢复状态信息

2、ViewConfiguration(常用常量mTouchSlop等)

3、ScaleGestureDetector 自定义辅助类

Android 屏幕分辨率适配

自定义View分辨率适配

学习中参考网上自定义 View 的例子，自己跟着写一遍，结果发现高分辨率模拟器上显示的跟想象中不一致，原来是没有做好分辨率适配

在自定义View中，由于 onDraw 方法中的 canvas 是使用 pix 作为单位的，因此可以在 init 方法中获取本机屏幕分辨率

在相关的 x 和 y 以及尺寸相关的处理中乘以 yDimen 或者 xDimen。

Layout中的百分比适配

详情参考鸿洋大神的开源库：
AndroidAutoLayout
